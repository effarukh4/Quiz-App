# Quiz-App

This is my beginner JavaScript project created while learning JavaScript fundamentals. It is a simple quiz application where users can answer multiple-choice questions and view their final score after completing the quiz.

## Features

- Multiple-choice questions
- Score calculation
- Dynamic question loading
- Displays the final score at the end of the quiz
- Simple and responsive interface

## Built With

- HTML
- CSS
- JavaScript

## Concepts Practiced

While building this project, I learned and practiced:

- Variables
- Arrays and Objects
- Functions
- Loops
- DOM Manipulation
- Event Listeners
- Conditional Statements
- Dynamic HTML Updates

## Project Structure


Quiz-App/
│── index.html
│── style.css
│── script.js
│── README.md


## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.
4. Read each question and select the correct answer.
5. Complete the quiz to view your final score.

## Learning Goal

The purpose of this project was to practice JavaScript fundamentals by building an interactive web application. It helped me understand how to work with arrays and objects, manipulate the DOM, handle user interactions through event listeners, and dynamically update webpage content based on user input.
